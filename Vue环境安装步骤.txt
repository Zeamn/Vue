一.安装nodejs（npm）    //node -v 查看版本

二.安装git或其他终端

三.可以使用淘宝镜像cnpm  //npm install -g cnpm --registry=https://registry.npm.taobao.org

四.安装vue脚手架 vue-cli //vue -V 查看脚手架的版本
//npm install vue-cli -g  (webpack才是不推荐安装到全局)

五.初始化项目
vue init <template-name>(有5个模版） <project-name>(项目名称)
//vue init webpack myVue
//这时就创建了一个myVue的文件夹了，里面就是常用的项目目录（脚手架）

六.进入项目文件夹(一定要进去项目文件夹后才能执行第七步)
cd myVue

七.安装项目所有的依赖包(package.json里指定的包),推荐用cnpm安装
npm/cnpm install

八.启动热更新
npm run dev

九.项目构建完毕，需要使用webpack打包
npm run build
