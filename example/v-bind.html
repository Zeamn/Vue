class与style的绑定

操作元素的 class 列表和内联样式是数据绑定的一个常见需求。 因为它们都是属性，所以我们可以用 v-bind 处理它们： 只需要通过表达式计算出字符串结果即可。 不过，字符串拼接麻烦且易错。因此，在将 v-bind 用于 class 和 style 时， Vue.js 做了专门的增强。表达式结果的类型除了字符串之外， 还可以是对象或数组。
1.可以直接绑定class样式
2.可以使用对象绑定class样式
3.可以使用对象绑定2个class样式
4.可以把对象整体放到Vue构造器的data里
1.可以使用数组绑定class样式,在data里找对应的值
2.数组三元表达式绑定class样式,在data里找对应的值
3.或者改写2为数组中内嵌对象
可以使用对象绑定style样式
直接绑定到一个样式对象
将多个样式对象应用到同一个元素上
